A responsive single-page Movie Explorer application built with HTML, CSS, and JavaScript using the OMDb API.

## Features
- Real-time movie search
- Dynamic content rendering
- Dark mode support
- Responsive design

## Setup
1. Get a free API key from https://www.omdbapi.com/
2. Replace `YOUR_OMDB_API_KEY` in `script.js`
3. Open `index.html` in a browser

Enjoy exploring movies!
